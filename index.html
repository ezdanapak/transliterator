QGIS Georgian Transliterator Plugin

Description: This QGIS plugin recognizes the Georgian alphabet and transliterates it into the Latin script in a new field.

Installation

1. Download the plugin from the GitHub repository.

2. Open QGIS and go to Plugins → Manage and Install Plugins.

3. Click Install from ZIP and select the downloaded file.

Usage

Once installed, the plugin provides an option to transliterate Georgian text in an attribute field to Latin.
